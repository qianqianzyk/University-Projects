# Nacl 项目管理系统

**软件工程实践课程设计 | 聚焦敏捷开发与项目管理**

## 项目简介

本项目是**软件工程课程设计**的产出成果，旨在实现一个聚焦于软件工程项目管理领域的综合性平台。项目基于 **Scrum** 和 **Kanban** 敏捷开发理念，通过调研分析业内主流工具（如 Taiga, Tenzu），设计并开发了一套适合中小型团队的敏捷项目管理系统

本项目严格遵循软件工程生命周期进行文档管理，所有设计文档均已整合至 `Nacl项目管理系统.pdf` 中

## 技术栈

本项目采用以下技术架构：

- **开发语言**: Python 3.x
- **后端框架**: Django 5.2
- **数据库**: MySQL / SQLite
- **前端技术**: HTML5 / CSS3 / JavaScript
- **开发方法**: 敏捷开发 (Agile/Scrum)

## 核心功能

### 1. 敏捷项目管理

- **Product Backlog**: 支持用户故事（User Stories）的创建、优先级排序和状态管理
- **Sprint Planning**: 迭代规划，支持将 Backlog 条目分配至特定的 Sprint
- **Kanban Board**: 可视化任务看板，支持拖拽流转任务状态（To Do -> In Progress -> Done）

### 2. 系统安全与审计

- **安全事件分析**: 实现了基于时间周期（周/月/年）的安全事件统计
- **登录审计**: 自动记录并分析失败的登录尝试，识别潜在的暴力破解行为 (`high_failed_logins`)
- **日志追踪**: 包含访问日志 (`AccessLog`) 和系统日志 (`SystemLog`) 的全量记录与查询

### 3. 用户与权限

- 基于角色的权限控制，区分管理员、项目经理、开发者等角色
- 支持用户组管理与权限分配
