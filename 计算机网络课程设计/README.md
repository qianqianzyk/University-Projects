# 计算机网络课程设计

本项目为2024/2025学年《计算机网络》课程设计，涵盖网络配置、协议分析、网络编程等多个任务模块，包含实验截图、配置文件、源代码及课程报告等内容。

## 目录结构说明

```
├── 任务1/                # 任务1：网络命令与协议分析，含实验截图
├── 任务2/                # 任务2：网络设备配置与集成，含Packet Tracer文件与拓扑截图
│   └── img/              #  任务2相关图片
├── 任务3/                # 任务3：网络编程与协议实现，含代码与运行截图
│   ├── ip/               #   任务3 C++源代码
│   └── img/              #   任务3相关图片
├── 网络编程源代码/        # 课程设计网络编程部分的完整源代码
│   ├── ip/               #   主代码
│   └── ip_copy/          #   代码副本
├── 网络配置/              # Packet Tracer网络配置文件
├── 课程设计报告.pdf       # 完整课程设计报告（PDF）
└── README.md             # 项目说明文件
```

## 各部分内容简介

- **任务1/**：包含ARP、IPCONFIG、NETSTAT、PING、TELNET、TRACERT等命令的实验截图，用于理解网络协议与命令行工具。
- **任务2/**：包含网络设备（路由器、交换机等）配置的Packet Tracer文件（.pkt）及拓扑、配置截图，展示网络集成与路由配置过程。
- **任务3/**：实现简单的网络协议编程（C++），并附有运行结果截图。
- **网络编程源代码/**：整理归档的网络编程相关C++源代码及其副本，便于查阅和复现。
- **网络配置/**：各类网络拓扑与配置的Packet Tracer文件，便于实验复现。
- **课程设计报告.pdf**：最终提交的课程设计报告。

## 使用说明

1. 推荐使用Cisco Packet Tracer打开`.pkt`文件进行网络配置实验。
2. C++源代码可在支持CMake的环境下编译运行，具体见`网络编程源代码/ip/CMakeLists.txt`。
3. 实验截图可用于报告撰写与结果分析。
