# 汇编语言程序设计实验集合

## 实验环境

- **汇编器**: MASM (或兼容的 8086 汇编器)
- **运行环境**: DOSBox 或 16 位 DOS 环境

## 实验内容

### 实验一：字符输入与大小写转换

任务描述：

编写一个程序，循环从键盘接收字符输入

- 如果输入的是**小写字母**（'a'-'z'），将其转换为**大写字母**并显示
- 如果输入的是其他字符，则原样显示
- 当用户按下 `ESC` 键（ASCII 1Bh）时，程序退出

### 实验二：十六进制数转二进制显示

任务描述：

编写程序，将内存中指定的十六进制数（代码中硬编码为 0ABCDH）转换为二进制形式并在屏幕上显示

- **输出格式**：二进制数每 4 位一组，组与组之间用空格隔开（例如：`1010 1011 1100 1101`）

### 实验三：响铃 N 次

任务描述：

编写程序实现蜂鸣器控制

- 从键盘输入一个数字字符 N（范围 '0'-'9'）
- 程序将输入字符转换为数值，并控制系统蜂鸣器（输出 ASCII字符 `07h`）响铃 N 次

### 实验四：查找数组中的最小偶数

任务描述：

在数据段定义一个包含 100 个字的数组 DATA（初始化为 100, 99, ..., 1）

- 编写程序遍历该数组
- 忽略数组中的所有奇数
- 在剩下的偶数中，找出**数值最小**的那个数，并将其打印在屏幕上

### 实验五：求解数组交集

任务描述：

在数据段定义两个字数组：A（包含 15 个奇数元素）和 $B$（包含 20 个连续整数元素）

- 编写程序遍历两个数组，找出它们共有的元素（即交集）
- 将找到的公共元素存储到数组 C 中
- 最后格式化打印数组 C 的所有内容

### 实验六：字符串大小写转换菜单

任务描述：

实现一个菜单驱动的字符串处理程序，提供以下选项：

1. **小写转大写**：输入一个字符串，将其中所有小写字母转换为大写，其他字符不变
2. **大写转小写**：输入一个字符串，将其中所有大写字母转换为小写，其他字符不变
3. **退出**：退出程序

### 实验七：0-100 格式化输出

任务描述：

编写程序，在屏幕上打印从 0 到 100 的所有整数

- **排版要求**：每行打印 10 个数字
- **分隔符**：数字之间用空格分隔

### 实验八：统计大小写字母个数

任务描述：

编写程序，接收用户输入的一个字符串

- 遍历字符串，分别统计其中**大写字母**（'A'-'Z'）和**小写字母**（'a'-'z'）的数量
- 分别输出两者的统计结果

### 实验九：求 N 个两位数的和

任务描述：

编写一个带有多重输入校验的求和程序：

1. **输入 N**：用户输入数字 N（范围 1-20）。如果输入错误超过 3 次，程序退出
2. **输入数值**：循环输入 N 个两位数（范围 10-99）。同样包含 3 次错误重试机会
3. **计算**：计算这 N 个数的总和并显示结果

### 实验十：数据类型判断与转换

任务描述：

接收用户输入的 4 个字符，根据输入内容执行不同逻辑：

- **二进制模式**：如果输入的 4 个字符全为 '0' 或 '1'，则将其视为一个 4 位二进制数，分别显示其**十进制值**和**十六进制值**
- **字符模式**：如果包含其他字符，则依次显示这 4 个字符的 **ASCII 码**（十六进制格式）
