# 电路与电子技术基础课程设计

## 项目简介

本项目是《电路与电子技术基础》课程的期末综合设计。项目包含两个核心部分的仿真与设计：

1. **直流稳压电源**：设计一个能够输出稳定直流电压的电源电路
2. **多功能信号发生器**：设计由运算放大器组成的信号发生电路，能够产生正弦波、方波、三角波、矩形波及锯齿波等多种信号

通过 NI Multisim 软件进行电路搭建与仿真，验证了变压、整流、滤波、稳压以及信号产生与处理的理论知识

## 系统模块与功能

本项目主要由以下两大系统组成：

### 1. 直流稳压电源系统

该模块旨在提供稳定的直流工作电压，包含以下四个环节：

- **电源变压器**: 将交流电网电压转换为合适的交流电压
- **整流电路**: 将交流电压转换为脉动的直流电压
- **滤波电路**: 滤除脉动直流中的交流成分
- **稳压电路**: 最终输出 **对称 ±12V 直流电压**，供后级运算放大器使用

### 2. 多功能信号发生器

该模块基于运算放大器设计，包含四种具体的波形发生电路：

- **正弦波信号发生器**: 产生标准的正弦信号
- **方波-三角波信号发生器**: 利用比较器和积分器电路产生方波和三角波
- **可调矩形波发生器**: 能够调节占空比的矩形波电路
- **可调锯齿波发生器**: 能够调节上升/下降斜率的锯齿波电路、

## 仿真结果与指标

在 Multisim 中运行仿真文件，可以观察到以下技术指标：

- **频率范围**: 信号发生器的频率可调，实验测得范围为 **16.6Hz (最低)** 至 **2.14kHz (最高)**
- **波形调节**:
  - 通过调节电位器（如 R20, R21），可以改变输出信号的频率
  - 通过调节相关电阻（如 R25），可以改变锯齿波的上下坡斜率
- **电源输出**: 稳定的 ±12V 直流电压

## 使用指南

1. **环境准备**: 安装 NI Multisim 14.0 或更高版本
2. **加载电路**: 打开 `仿真综合设计.ms14`
3. **运行仿真**: 点击运行按钮 (F5)
4. **观察波形**:
   - 双击电路图中的**示波器 (XSC1)**，观察正弦波、三角波或锯齿波的形状与频率
   - 调节电路中的**可变电阻 (Potentiometer)**，观察波形频率和占空比的实时变化

## 总结

本项目系统地实践了电子电路设计的完整流程。从整流滤波到信号产生，深入理解了三端集成稳压器和运算放大器的工作原理。通过解决频率变化伴随的波形失真等问题，优化了电路参数，实现了预期的设计指标
