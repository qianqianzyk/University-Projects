# 智能多功能小车工程实践项目

本项目包含两个主要子系统：
- **VoiceSystem**：基于树莓派的全流程语音控制多功能坦克车
- **FaceTracking**：基于CS架构的视频流与人脸追踪系统

适用于浙江工业大学计算机工程实践课程，支持多种智能交互与远程协作。

## VoiceSystem 主要功能

- 语音识别与指令分发（百度云API）
- 通义千问大模型AI对话
- 小车底盘运动（前进/后退/转向/跳舞等）
- 超声波避障、红外循迹（均支持语音中断）
- 摄像头图像识别、手势识别、二维码识别
- 天气查询与语音播报
- 人脸检测与邮件报警
- 七彩LED与舵机灯光秀

**典型指令举例**：
| 指令/功能           | 描述                                                         |
|--------------------|--------------------------------------------------------------|
| 向前/后/左/右移动   | 基础动作，控制小车运动                                       |
| 原地左/右转         | 原地旋转                                                     |
| 立即停止           | 停止运动                                                     |
| 沿着黑线走         | 红外循迹，语音“中断”可退出                                   |
| 避开障碍物         | 超声波避障，语音“中断”可退出                                 |
| 跳舞               | 小车跳舞，配合LED与舵机动作                                  |
| 查询今日天气       | 爬虫获取杭州天气，语音播报                                   |
| 跟踪敌人           | 人脸检测，抓拍并邮件报警，语音“中断”可退出                   |
| 启动二维码识别     | 摄像头识别二维码，解码内容                                   |
| 启动手势识别       | 摄像头识别手势，识别到“六”退出                               |
| 启动图像识别       | 摄像头拍照并识别物体                                         |
| 给大家打个招呼     | LED灯光秀+语音                                               |
| 小车+任意问题      | 通义千问大模型AI对话，语音播报                               |

**详细用法、依赖与扩展说明请见** [`VoiceSystem/README.md`](VoiceSystem/README.md)

## FaceTracking 主要功能

- CS（Client-Server）架构，支持树莓派与PC端协同
- MJPG-Streamer视频流服务，实时摄像头画面传输
- 客户端支持人脸检测与追踪
- 典型应用：远程监控、目标跟踪、与小车系统协作

**使用流程简述**：
1. 树莓派上启动MJPG-Streamer，保证摄像头正常
2. 客户端配置端口号与流地址，确保与服务端一致
3. 先启动服务器端，再启动PC客户端，建立通信

**常见问题与解决方案、详细说明请见** [`FaceTracking/README.md`](FaceTracking/README.md)

## 系统协作与扩展

- 两大子系统可独立运行，也可协同用于复杂场景（如语音控制+远程视频监控）
- 推荐在局域网环境下部署，保证网络与音视频流畅
- 支持多线程、模块化扩展，便于二次开发
